**PyScan** is a fast, multi-threaded port scanner built for red team operations.  
Inspired by Nmap but lightweight and customizable.

## 🚀 Features

- ⚡ Multi-threaded scanning (100+ threads)
- 🎯 Service detection on common ports
- 📡 Banner grabbing
- 💾 Save results to file
- 🎨 Colorized output
- 📦 Zero external dependencies

## 📦 Installation

```bash
git clone https://github.com/YOUR_USERNAME/pyscan
cd pyscan
chmod +x install.sh
./install.sh
